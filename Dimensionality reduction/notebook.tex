    \section{Principle Componet Analysis}\label{principle-componet-analysis}

    \texttt{Principal\ component\ analysis\ (PCA)} is a statistical
procedure that uses an orthogonal transformation to convert a set of
observations of possibly correlated variables into a set of values of
linearly uncorrelated variables called principal components. If there
are \texttt{n} observations with \texttt{p} variables, then the number
of distinct principal components is \texttt{min(n-1,p)}. This
transformation is defined in such a way that the first principal
component has the largest possible variance (that is, accounts for as
much of the variability in the data as possible), and each succeeding
component in turn has the highest variance possible under the constraint
that it is orthogonal to the preceding components. The resulting vectors
are an uncorrelated orthogonal basis set. PCA is sensitive to the
relative scaling of the original variables.

    \subsubsection{PCA降维}\label{pcaux964dux7ef4}

\begin{itemize}
\tightlist
\item
  目标：提取最有价值的信息（基于方差）
\item
  问题：降维后数据失去原来意义（可以保密） 
\end{itemize}

\subsubsection{原理：}\label{ux539fux7406}

\begin{itemize}
\tightlist
\item
  向量、基：基是正交的（相互垂直、线性无关）
\item
  基变换：空间变换 
\end{itemize}

\subsubsection{基变换依据：}\label{ux57faux53d8ux6362ux4f9dux636e}

\begin{itemize}
\tightlist
\item
  协方差矩阵：寻找一个一维基，使得所有数据变换为这个基上的坐标表示后，方差值最大.
  直观：投影后的投影值尽可能的分散
\item
  方差： \(Var(a)=\frac{1}{m}\sum_{i=1}^m (a_{i}-{\eta})^2\)
\item
  协方差(假设均值为0):
  \(Cov(a,b)=\frac{1}{m}\sum_{i=1}^m (a_{i}b_{i})^2\)
\item
  说明：协方差为0表示两个字段完全独立，表现为两个基正交 
\end{itemize}

\subsubsection{优化目标：}\label{ux4f18ux5316ux76eeux6807}

\begin{itemize}
\tightlist
\item
  将一组N维向量降为K维，目标是选择K个单位正交基，使原始数据变换到这组基上后，各字段两两间协方差为0，字段的方差尽可能大
\item
  \texttt{协方差矩阵}：对角线元素为两个字段的方差，其他元素为协方差
\item
  数学原理：\texttt{特征值、特征向量}
\item
  采取措施：协方差矩阵\texttt{对角化}：除对角线外，其他元素化为0，并且对角线上元素按大小从上到下排序
\item
  数学定理：实对称矩阵（一个n行n列的实对称矩阵一定可以找到n个单位正交特征向量）------实对称阵对角化
\end{itemize}
